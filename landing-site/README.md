# Al-Ansaar Landing Site

Public landing page for **Al-Ansaar** (الانصار) — readiness training for Imam al-Asr.

## Run

```bash
cd landing-site
npm install
npm run dev
```

## Download URL

Edit `lib/site.ts` → `GITHUB_RELEASES_URL` when the final GitHub Releases / APK link is ready (currently points at the releases page).

## Stack

Next.js 16 · Tailwind CSS v4 · Framer Motion · Aref Ruqaa / Source Serif 4
